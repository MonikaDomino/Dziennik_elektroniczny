1.Otwieramy program Netbeans.
2.Otwieramy projekt StudenciGui.
3.Dodajemy do Library pliki:
	- r2xml.jar
	- derbyclient.jar
4.Następnie łączymy się z bazą.
5.Klikamy na pasku menu Window -> Services
6.Otwieramy Databases
7.Klikamy na JavaDB i łączymy się z bazą
8.Następnie gdy już się połączymy tworzymy baze danych
9. Klikamy Create Databases
10. Uzupełniamy:

Database Name: APP
username: app
password: test

11.Następnie łączymy się z tą bazą i odpalamy aplikacje GUI.
12. W aplikacji może dodawać studenta do bazy, usuwać studenta podając jego ID
oraz modyfikować nazwisko konkretnego studenta o określonym ID.
